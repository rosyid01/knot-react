### Boilerplate of Golang + Reactjs + React Router + LESS

yay

### Start the UI

```bash
cd $GOPATH/src/github.com/novalagung/knot-react/webapp/view
npm install
npm start
```

### Build UI for Production

```bash
cd $GOPATH/src/github.com/novalagung/knot-react/webapp/view
webpack
```

### Run the Application

```bash
cd $GOPATH/src/github.com/novalagung/knot-react/webapp
go run main.go
```
